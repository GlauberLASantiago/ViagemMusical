# Viagem Musical - Montanha-Russa 3D

Uma experiência imersiva de montanha-russa tridimensional construída com **Three.js**, onde o trilho é um piano gigante e o cenário é um cosmos musical repleto de planetas em formato de notas.

## 🚀 Funcionalidades

- **Trilho de Piano Interativo**: Percorra uma pista sinuosa composta por teclas de piano (brancas e pretas) geradas proceduralmente seguindo um padrão musical real.
- **Cenário Cósmico**: Navegue por um espaço profundo com 441 planetas esféricos 3D que possuem gravuras de figuras musicais (semínimas, colcheias, claves, etc.) e giram em seus próprios eixos.
- **Pentagrama de Vidro**: Um conjunto de linhas musicais (pentagrama) em material de vidro translúcido que serpenteia o cenário em uma escala monumental.
- **Efeitos Visuais Premium**:
  - Neblina (Fog) volumétrica para dar profundidade.
  - Partículas de poeira estelar para sensação de velocidade.
  - Notas musicais em estilo "Billboard" que flutuam e reagem à proximidade da câmera.
  - Iluminação dinâmica com faróis coloridos (ciano e magenta).

## 🎮 Controles

- **Seta para Cima**: Aumentar velocidade.
- **Seta para Baixo**: Reduzir velocidade.
- **Seta para Esquerda / Direita**: Movimentação lateral manual no trilho.
- **Mouse**: Controle fino do deslocamento lateral (offset).

## 🛠️ Tecnologias Utilizadas

- **HTML5** & **CSS3** (Vanilla)
- **Three.js** (WebGL)
- **Javascript** (ES6+)

## 📦 Como Rodar o Projeto

1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente em qualquer navegador moderno (Chrome, Firefox, Edge, etc.).
3. *Dica*: Para melhor performance, utilize um navegador com aceleração de hardware ativada.
